Class ActsAsXapian::Search
In: vendor/plugins/acts_as_xapian/lib/acts_as_xapian.rb
Parent: QueryBase

Search for a query string, returns an array of hashes in result order. Each hash contains the actual Rails object in :model, and other detail about relevancy etc. in other keys.

Public Class methods

new(model_classes, query_string, options = {})

model_classes - model classes to search within, e.g. [PublicBody, User]. Can take a single model class, or you can express the model class names in strings if you like. query_string - user inputed query string, with syntax much like Google Search
